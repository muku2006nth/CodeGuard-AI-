# Google Colab — Training Only

Open **`train_codebert.ipynb`** in [Google Colab](https://colab.research.google.com/).

1. **Runtime → Change runtime type → T4 GPU**
2. Clone repo (`REPO_URL`) or upload your **project code** zip
3. Run all cells — **DetectVul/devign** downloads from Hugging Face automatically
4. Download `codebert-vuln.zip` and `metrics.json`
5. On PC: extract to `models/codebert-vuln/` and `results/metrics.json`

Full workflow: [docs/WORKFLOW.md](../docs/WORKFLOW.md)
